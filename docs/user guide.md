# Generator Winding Visualizeer User Guide
## Core Concepts
**Winding Layout**
IN the middle in *CAD CANVAS* the user can find the Winding Layout where they can change the phase and direction of the winding for each slot and position to wind the generator. 

**Settings**
Under *Settings* the user can change the number of Phases, Poles, Positions and Slots to reconfigure the winding layout. E.G. having a three phased or five phased generator. The generator also operates under a certain RPM which can be changed as well as how the field from the magnet changes by choosing a Magnet function.

**Overview**
In *Overview* information about the number of windings for each phase can be found as well as a voltage diagram over of the generator in action.

**Overtones**
Under *Overtones* the user can find a table wind information about the different harmonics which the output Phase Voltage wave can be decomposed into for each phase. Furthermore the user can see the THD (Total Harmonic Distortion) stat.

## Using the program
1. **Configuration:** Configure all settings under *Settings*. 
2. **Processing:** Wind the generator in the *Winding Layout*. 
3. **Simulating:** Click the *Simulate* button to run the simulation with your winded generator.
4. **Reviewing:** Analyse the result firstly in Overview to get a grasp of the overall performance and then in Overtones for further deep diving.
